# Cerebro

# Using API's 
 - https://www.instagram.com/developer/
 - https://tech.yandex.ru/maps/jsapi/

# You will need 
 - python3

# How to start

git clone

./server.py

Then launch cerebro.html in browser, click on map and enjoy realtime photos from this place!
